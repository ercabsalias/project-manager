# 🧑‍💼 Project Manager
Aplicação moderna de gerenciamento de projetos

Criei uma aplicação web elegante e intuitiva para gerenciar projetos, tarefas e equipes de forma eficiente. Ideal para times, freelancers ou qualquer pessoa que deseje acompanhar seu fluxo de trabalho com estilo.

# ✨ Funcionalidades Principais

✅ Design System Moderno
Cores personalizadas, gradientes suaves e tokens semânticos para manter consistência visual e escalabilidade.

✅ Dashboard Intuitivo
Painel inicial com visão geral, estatísticas e acesso rápido aos projetos mais recentes.

✅ Gestão de Projetos
Lista dinâmica de projetos com filtros avançados e cards interativos.

✅ Detalhes do Projeto
Página dedicada para cada projeto com tarefas organizadas e informações da equipe envolvida.

✅ Mock API Integrada
Simulação de dados (projetos, tarefas, membros) para facilitar testes e desenvolvimento.

✅ Interface Responsiva
Experiência fluida em todos os dispositivos (desktop, tablet e mobile).

✅ Navegação Suave
Sidebar fixa com roteamento entre páginas para navegação ágil.

✅ Autenticação de Interface (Frontend)
Tela completa de login e registro com suporte a OAuth, hero image e design atrativo.

A aplicação está pronta para ser expandida com autenticação, drag & drop, notificações e outras funcionalidades avançadas.

# 🚀 Começando
Clone o repositório:
```bash
git clone https://github.com/ercabsalias/project-manager.git
cd project-manager
```
Instale as dependências:
```bash
yarn install
```

Inicie o ambiente de desenvolvimento:
```bash
yarn dev
```
